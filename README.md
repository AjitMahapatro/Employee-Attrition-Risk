### üõ† Project Overview
This project integrates **Python Machine Learning** with **Power BI** to predict employee churn. It quantifies financial risk ($11M exposure) and identifies key factors driving attrition.

**Tech Stack:**
* **Python:** Random Forest Classifier for risk scoring.
* **MySQL:** Database for managing employee records.
* **Power BI:** Interactive dashboard for HR stakeholders.

### üîç Key Insights
* **Risk Factors:** Identified Overtime and Monthly Income as the top predictors of churn.
* **Financial Impact:** The model flagged high-risk employees, allowing for interventions that saved a projected **$474K**.
